# Manager 系統生產環境部署方案

## 架構概述

```
用戶 ──► Cloudflare CDN ──► Cloudflare Pages (前端)
                        ──► EC2 Instance (後端 API)
```

## 部署規劃

### 前端：Cloudflare Pages
- React 靜態檔案部署
- 自動 HTTPS 和全球 CDN

### 後端：AWS EC2
- Docker 容器化部署
- Nginx 反向代理
- API 服務

### SSL 憑證
- 使用 Cloudflare Origin Certificate
- 或 Let's Encrypt 自動憑證

## 域名配置
- 前端：`manager.yourdomain.com`
- 後端：`api.manager.yourdomain.com`
